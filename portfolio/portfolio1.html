Vending Machine Application

The Vending Machine application is a command line application that allows the user to view a list of items for sale inside of the machine, insert money to purchase the item selected, and then logs the transaction as well as giving back the correct amount of change. The items quantity that is available is adjusted according to how many the user chooses to purchase.
